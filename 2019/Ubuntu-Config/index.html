Ubuntu 19.10 Config

Veröffentlicht am 2019-12-10 in Debug & Config Aufrufe:
Symbole im Artikel gezählt: 615 Wörter Lesezeit ≈ 1 min

今天大半天全都用来调试 Ubuntu 了。之前因为系统从 18.04 直接升级到 19.10，很多地方不稳定，还有无尽的 dependency issues， 今天索性把所有东西重装一遍。

​

Ubuntu 19.10 的界面虽然作了更新，但系统自带的几款主题仍然巨丑无比。于是把之前一直心水不已的 Kali 主题移植过来，瞬间爱了～

图里分别是 Files， Terminal， 和 Gedit。Gedit 自带的皮肤仍然是配色非常迷，因此自己参照版式，写了一个配色方案。近似于 One Dark，但配色更符合个人口味。

Terminal 修改系统配置文件，加了个颜文字，哈哈～


剩下的就是各种配置了。之前的系统打包复制了一些东西（dump），但有的还是重新再来一遍 （旧系统的配置已经乱七八糟）。

于是进行了如下配置：

Name Objective Commands / Notes
MySQL 解除 sudo mysql -uroot
Docker 解除 sudo docker run hello-world
JDK 版本依赖 sudo update-alternatives --config java
Jenkins (8) Debug 血与泪的教训。。。关于第三方 Plugins 的安装
Tomcat Debug 系统路径的配置，Catalina Debug
Anyconnect Debug 旧版本无法启动
Pinyin Debug 换掉 iBus 自带的拼音，用 Intelligent Pinyin

其它就不一一列举了。以后有新的配置，会进行更新。
